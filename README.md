# 🐾 宠物带货公众号内容仓库

> 专注宠物领域优质内容创作，打造有温度、有干货、有转化的公众号文章矩阵。

## 📖 项目简介

本仓库包含 **200 篇** 宠物领域公众号文章，涵盖宠物食品、宠物用品、宠物健康、宠物时尚四大品类。每篇文章均采用公众号爆款风格撰写，兼具科普价值与带货能力。

### 内容特色

- 🔥 **标题吸睛** — 每个标题都经过精心设计，适合公众号传播
- 📚 **科普实用** — 内容真实有深度，为读者提供真正有价值的信息
- 🛒 **自然带货** — 每篇推荐 2-3 款产品，融入正文不生硬
- 🏷️ **标签体系** — 方便内容管理和 SEO 优化

## 📝 文章格式说明

每篇文章遵循统一格式：

```markdown
# 标题

> 引言/导语（吸引读者继续阅读）

正文内容（800-1200字，分段清晰，配小标题）

## 🛒 好物推荐

| 产品 | 亮点 | 参考价 | 购买链接 |
|------|------|--------|---------|
| 产品名 | 核心卖点 | ¥XX | [点击购买](#) |

---
标签：#标签1 #标签2 #标签3
```

## 🚀 使用方式

1. 浏览 `articles/index.md` 查看全部文章列表
2. 按分类找到感兴趣的文章编号
3. 阅读或编辑对应 `.md` 文件
4. 替换购买链接占位符为实际链接即可发布
